# Notes

## How to use the REPL
1. ctrl + opt + c, then ctrl + opt + j, then jack in to Leiningen
2. Open file to use
3. ctrl + opt + c, then Enter to load file into REPL

Tips:
- Use comment blocks to store test code and data
- Use ctrl + Enter to evaluate a snippet
